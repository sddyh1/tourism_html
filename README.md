# TravelWorld — туристическое агентство

<img width="1892" height="963" alt="image" src="https://github.com/user-attachments/assets/5a3126cf-3d7f-4006-adff-7eacd206419c" />

Лендинг туристического агентства — лабораторная работа по HTML и CSS.  
Чистый стек без фреймворков: вёрстка и стили написаны вручную.

## Возможности

- Hero-блок с фоновым изображением и кнопкой призыва к действию
- Информационный блок «О нас»
- Карточки популярных направлений (Италия, Швейцария, Мальдивы) с ценами
- Плавная прокрутка к якорям при клике по пунктам навигации
- Адаптивная вёрстка: на узком экране шапка перестраивается, карточки уходят в колонку
- Эффекты при наведении (кнопки, карточки, изображения)
- Локальные изображения — все картинки загружаются из папки `images`

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — переменные, Flexbox, медиа-запросы, анимации
- **Google Fonts** — шрифты Montserrat

## Запуск

Достаточно открыть `index.html` в любом современном браузере.

> **Примечание:** перед запуском поместите свои изображения в папку `images` с указанными выше именами. При необходимости измените пути в `index.html` и `style.css`.

## Архитектура

Проект построен по классической схеме статического сайта:

- **HTML** — отвечает за структуру и контент
- **CSS** — управляет внешним видом и адаптивностью
- **Изображения** — хранятся локально, что ускоряет загрузку и не зависит от внешних сервисов
